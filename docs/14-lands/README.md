# 14 — Lands (Terrenos)

> **Status: EM BREVE** — sem código hoje. Registrado aqui o desenho econômico alvo para a fase de expansão do mundo.

## Visão

Terrenos são parcelas NFT do mapa-múndi com **produção passiva limitada e custo de manutenção ativo** — desenhados para NÃO repetirem o erro clássico de land = impressora de dinheiro.

## Desenho resumido

- **Oferta:** coleção limitada por região, vendida em DOL (parte queimada, parte treasury) em ondas pequenas conforme a base de jogadores cresce — nunca land antes de demanda.
- **Produção:** terreno gera **materiais de craft** (não GOLD direto) coletados por visita; produção decai sem manutenção.
- **Manutenção:** custo semanal em GOLD (sink recorrente).
- **Melhorias:** construções (mina, herbário, posto de caça) compradas com GOLD+materiais aumentam o tipo/qualidade da produção.
- **Social:** dono pode abrir o terreno a visitantes cobrando pedágio em GOLD (P2P com taxa de protocolo).

## Regras anti-inflação

1. Land nunca emite token — só materiais, que precisam de forja/alquimia (mais sinks) para virarem valor.
2. Manutenção > 0 sempre: terreno abandonado para de produzir.
3. Oferta gateada por MAU: nova onda só quando `jogadores ativos / terrenos > limiar`.
