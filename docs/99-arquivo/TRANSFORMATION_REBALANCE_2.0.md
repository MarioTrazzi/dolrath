## 🎮 **NOVAS ESTRATÉGIAS**

### **⚡ Para Raças com Transformação:**

1. **Timing de Transformação:**
   - Início: Máximo aproveitamento da duração
   - Meio: Quando ficar em desvantagem
   - Nunca: Se já dominando sem transformar

2. **Gestão de Recursos:**
   - MP baixo, mas ainda importante
   - Stamina para outras ações
   - Cooldown planejado

3. **Setup Defensivo:**
   - Primeiro turno = vulnerável
   - Posicionamento estratégico
   - Defesa antes de transformar

### **🛡️ Para Raças sem Transformação:**

1. **Pressão Imediata:**
   - Ataque agressivo no turno 1
   - Impedir setup de transformação
   - Forçar decisões difíceis

2. **Sustain Superior:**
   - Stats constantes = previsibilidade
   - Sem cooldowns = flexibilidade
   - Sem custos = mais ações

3. **Aproveitamento de Janelas:**
   - 1 turno de transformação = oportunidade
   - Cooldowns = momentos de domínio
   - Recursos gastos = vantagem
