## 🎮 Como Jogar Agora

### 1. Acesse o Jogo
```
http://localhost:3000
```

### 2. Navegação
- **Homepage**: `/` - Visão geral do sistema
- **Criar Personagem**: `/character` - Customize seu herói
- **Arena de Combate**: `/combat` - Lute contra inimigos

### 3. Comandos de Combate
Digite no chat durante o combate:
- `[ataque]` - Ataque básico
- `[defender]` - Posição defensiva (+defesa)
- `[esquivar]` - Preparar esquiva (+agilidade)
- `[contra-ataque]` - Arriscado, mais dano

### 4. Exemplo de Combate
```
🎮 Você: [ataque]
🤖 IA: Gorak o Destemido desfere um golpe preciso brandindo 
       sua arma com maestria contra Drako Flamejante!
       💥 Dano: 8 ❤️ HP: 142/150
       🎲 7 + 4 = 11 🎯
```
