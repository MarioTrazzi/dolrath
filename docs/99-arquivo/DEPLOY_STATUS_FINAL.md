## 🎯 **DIFERENÇAS ANTES vs DEPOIS:**

### ❌ **ANTES (Desbalanceado):**
- Meta único: Todo mundo STR
- INT inútil (sem magia no PvP)
- AGI fraco (pouco retorno)
- DEF só dava +2 HP

### ✅ **AGORA (Balanceado):**
- 4 builds viáveis diferentes
- Sistema de magia funcional
- Críticos e esquiva baseados em AGI
- DEF reduz dano real recebido
- Transformações criam variedade
